# Tiny URL - 短地址服务

## 项目简介

Tiny URL 是一个基于 C 语言实现的控制台版短地址服务项目。

本项目能够将较长的 URL 映射为固定长度（5 位 Base62 编码）的短地址，并支持短地址解析、访问统计、访问次数限制等功能，采用模块化开发方式实现，适合作为 C 语言课程设计或数据结构综合实验项目。

---

## 项目功能

* 生成短地址（Create Short URL）
* 根据短地址解析原始 URL（Parse Short URL）
* 查看所有短地址记录
* 统计每个短地址的访问次数
* 设置短地址最大访问次数
* 判断短地址是否失效
* 控制台菜单交互（CLI）

后续可扩展功能：

* 数据文件保存（程序退出后数据不丢失）
* 自定义短地址
* 设置有效期限
* 删除短地址
* 哈希表优化查询效率
* 网络版本（HTTP API）

## 数据结构

每条短地址记录包含以下信息：

* 自增 ID
* 原始 URL
* 短地址
* 当前访问次数
* 最大允许访问次数
* 是否有效

所有记录保存在内存数组中进行管理。

---

## 核心算法

### 1. 自增 ID

系统维护一个全局自增编号。

```text
1
2
3
4
...
```

每创建一个新的 URL，自动分配一个新的 ID。

---

### 2. Base62 编码

采用 Base62 字符集：

```text
0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz
```

例如：

```text
10  -> A
35  -> Z
36  -> a
61  -> z
62  -> 10
```

将十进制 ID 转换为固定 5 位 Base62 字符串：

```text
1   -> 00001
35  -> 0000Z
61  -> 0000z
62  -> 00010
```

---

### 3. 短地址解析

输入短地址后：

1. 在数据库中查找对应记录；
2. 判断是否存在；
3. 判断是否超过最大访问次数；
4. 返回原始 URL；
5. 更新访问统计。
